# Code of Conduct

SporTagLytics は、建設的で安全なコラボレーションを前提に運営します。すべての参加者は、issue、pull request、review、discussion、ドキュメント、AI 生成提案を含むすべての場でこの方針に従ってください。

## Expected Behavior

- 技術的な主張は、再現手順、根拠、トレードオフを添えて説明する。
- 人ではなく、コード、設計、ドキュメント、運用上の問題を批評する。
- 異なる経験、言語、専門領域、利用環境を尊重する。
- セキュリティ問題や個人情報を public issue に書かない。
- AI 生成物を提出する場合も、提出者が内容をレビューし責任を持つ。

## Unacceptable Behavior

- ハラスメント、差別、脅迫、個人攻撃。
- 個人情報や非公開情報の公開。
- 明確な悪意を持つスパム、荒らし、故意のミスリード。
- セキュリティ脆弱性の悪用手順を不必要に公開すること。

## Enforcement

メンテナは、問題のある投稿の編集依頼、非表示、ロック、参加制限などの対応を行うことがあります。深刻なセキュリティ問題は [SECURITY.md](SECURITY.md) の手順で扱います。
