# SporTagLytics

スポーツ映像（ラグビー想定）のタグ付け・統計可視化を行うElectronデスクトップアプリ。

- 2映像の同期再生（プライマリ/セカンダリを角度情報から自動割り当て、オフセットは手動/再分析で更新）
- ラベルグループ付きのタグ付け（カスタムレイアウトのコードウィンドウ・アクションリンク・ホットキーを設定可能）
- ビジュアルタイムライン編集（ズーム、範囲選択、重複/移動/ラベル付与、プレイリストへの追加、インライン編集）
- 統計ダッシュボード（ポゼッション/結果/種別/モメンタム/クロス集計、映像ジャンプ対応）
- タイムラインとクリップのエクスポート/インポート（JSON/CSV/SCTimelineフォーマット、オーバーレイ付きクリップ書き出し）
- プレイリスト専用ウィンドウ（連続/ループ再生、メモ、フリーズフレーム、簡易描画）
- 設定画面でテーマ・オーバーレイ・アクションプリセット・ホットキー・コードウィンドウを管理（未保存検知付き）

## 使い方

### 1. 初回起動
- オンボーディングチュートリアルが自動表示されます（スキップ可）。
- ウェルカム画面でフォルダをドラッグ&ドロップ、または「既存パッケージを開く」「新規パッケージを作成」を選択。
- 最近開いたパッケージを一覧から再オープン可能（メニューの履歴とも連動）。

### 2. パッケージを開く
- 「既存パッケージを開く」またはフォルダをドラッグ&ドロップ。
- `.metadata/config.json` を読み込み、角度リスト（`angles`）からプライマリ/セカンダリの映像パス・チーム名・同期情報を復元します（パスは相対パスに自動変換）。
- 音声同期データが無い場合はそのまま再生されるので、必要に応じて後述の音声同期を実行してください。

### 3. 新規パッケージ作成
4ステップウィザードで作成（基本情報 → 保存先 → 映像選択 → 確認）。
- 1つ以上のアングルを登録可能（1番目がプライマリ、2番目がセカンダリとして初期割り当て）。
- 生成物: `.metadata/config.json`（angles/primary/secondary/syncData を含む）, `timeline.json`, `videos/`（指定ファイルをコピー＆リネーム）。
- 作成直後は音声同期を実行しません。必要に応じてメニュー/ホットキーで同期してください。

### 4. 音声同期と同期モード
- メニュー/ホットキーで操作  
  - `Cmd/Ctrl + Shift + S`: 音声同期を再実行（進捗バー付き）  
  - `Cmd/Ctrl + Shift + R`: 同期リセット（オフセット0）  
  - `Cmd/Ctrl + Shift + M`: 現在位置で手動同期（2映像の時間差をその場で採用）  
  - `Cmd/Ctrl + Shift + T`: 手動モード切替（オン時は各プレイヤーの個別シークを許可）
- オフセットは `.metadata/config.json` に保存され、再生時に自動適用されます（同期進捗はバックドロップで表示）。

### 5. タグ付け（コードパネル）
- チームごとにアクションボタンを表示。クリックで録画開始/終了し、同じ時間帯で複数アクションの同時記録も可能。
- ラベルグループ構造（`groups`）に対応。ラベルを選択した状態で記録すると `labels` 配列として保存（`actionType/actionResult` は後方互換の自動補完）。
- ホットキーはアクションごとに設定でき、2チーム目は Shift+同じキーで起動。
- コードウィンドウは設定画面で自由配置レイアウトを作成可能（ボタン色/サイズ/リンク、`${Team1}/${Team2}` プレースホルダー、コード/ラベルモード切替）。

### 6. タイムライン編集
- ビジュアルタイムラインでイベントを表示/編集。  
  - クリックでジャンプ、Cmd/Ctrl+ホイールでズーム、ドラッグで範囲選択。  
  - コンテキストメニューから編集/削除/ジャンプ/重複/プレイリスト追加。  
  - Enter で編集ダイアログ、Delete/Backspace で削除、Tab/Option+↑↓で同一アクション内を移動。  
  - 編集ダイアログで時刻・メモ・ラベルをまとめて更新、複数選択に一括ラベル付与ダイアログあり。  
- 変更は `timeline.json` に自動保存され、Undo/Redoにも対応。

### 7. 統計ダッシュボード（分析 → `Cmd/Ctrl + Shift + A`）
- **ポゼッション**: 時間集計（円グラフ）
- **アクション結果/種別**: アクション別に結果/種別を集計（円グラフ）
- **モメンタム**: ポゼッションの流れを棒グラフ表示（色分け: Try/Positive/Negative/Neutral）
- **クロス集計**: 任意の行/列軸（チーム・アクション・ラベルグループ/all_labels）でヒートマップ表示  
  - チーム/アクション/ラベルでフィルタ  
  - セルをクリックすると該当イベントへジャンプ

### 8. エクスポート/インポート
- メニュー「ファイル > エクスポート」で操作。  
  - タイムライン: JSON（内部形式）、CSV（`時刻, 終了時刻, アクション名, タイプ, 結果, 備考`）、SCTimeline互換JSON（SCTimeline）  
  - クリップ書き出し: 選択または全体を FFmpeg で出力。単一/インスタンスごと/行ごと、1 or 2アングル結合、ファイル名プレフィックス、オーバーレイ表示（設定のテンプレートと連動）を指定可能  
- メニュー「ファイル > インポート」で JSON/SCTimeline を読み込み（JSON優先で自動判定）。

### 9. プレイリストウィンドウ
- タイムラインの複数選択をプレイリストに追加（右クリックメニューまたはツールバーのプレイリストボタン）。
- プレイリストは複数管理でき、順番変更・メモ・ループ（なし/単一/全体）を設定可能。
- ヘルプメニュー/プレイリストボタンから専用ウィンドウを開き、プレイリストの再生・フリーズフレーム・簡易描画・ノート編集が可能。プレイリストからメインプレイヤーへのシーク/再生も行えます。

### 10. 設定
- メニュー「設定」 (`Cmd/Ctrl + ,`) で開く（未保存変更を検知）。  
  - 一般: テーマ（ライト/ダーク/システム）、クリップ書き出しオーバーレイの表示項目/テンプレート  
  - アクションプリセット: 追加/編集/削除、ラベルグループ/ホットキー設定、プリセットのインポート/エクスポート（重複IDは自動リネーム）  
  - ホットキー: 再生・同期・分析・Undo/Redo などのキーバインド編集（衝突・禁止キーを検知、デフォルトにリセット可）  
  - コードウィンドウ: レイアウトの新規/複製/削除、ボタン配置とリンク（排他/連動/無効化）、グリッドスナップ、`${Team1}/${Team2}` プレースホルダー、レイアウトのエクスポート/インポート
- 変更はローカル設定ファイルに保存され、アプリ全体に反映されます。

## データ構造

### パッケージ
```
PackageName/
├── .metadata/
│   ├── config.json          # チーム名、アクションリスト、角度情報(angles, primary/secondary)、syncData(syncOffset など)
│   └── sync.json (未使用)   # 互換性維持用
├── timeline.json            # タイムラインデータ（自動保存・Undo対応）
└── videos/
    ├── Angle 1.mp4          # angles[].relativePath と一致（ウィザードでコピー＆リネーム）
    └── Angle 2.mp4 ...      # 2本目以降は任意（再生には主にプライマリ/セカンダリを使用）
```

### タイムライン
```ts
type TimelineData = {
  id: string;               // ULID
  actionName: string;       // 例: "TeamA Lineout"
  startTime: number;        // 秒
  endTime: number;          // 秒
  qualifier: string;        // メモ
  labels?: { name: string; group?: string }[]; // ラベルグループ構造
  color?: string;           // コードウィンドウのボタン色（未指定時はチーム色）
  // 非推奨: 後方互換フィールド（labelsが無い場合に自動補完）
  actionResult?: string;
  actionType?: string;
};
```

## デフォルトホットキー

| キー                         | 機能                     |
| ---------------------------- | ------------------------ |
| `Space`                      | 再生/停止                |
| `Right` / `Shift+Right`      | 0.5倍速（スロー） / 2倍速再生（押下中のみ速度変更、キーアップで1倍に戻る） |
| `Cmd/Ctrl+Right` / `Option+Right` | 4倍速 / 6倍速再生（押下中のみ） |
| `Left` / `Shift+Left`        | 5秒戻し / 10秒戻し       |
| `Cmd/Ctrl+Shift+S`           | 音声同期を再実行         |
| `Cmd/Ctrl+Shift+R`           | 同期をリセット           |
| `Cmd/Ctrl+Shift+M`           | 今の位置で同期（手動）   |
| `Cmd/Ctrl+Shift+T`           | 手動モード切替           |
| `Cmd/Ctrl+Shift+A`           | 統計ダッシュボード       |
| `Cmd/Ctrl+Z` / `Cmd/Ctrl+Shift+Z` | Undo / Redo       |
| アクションプリセットで設定   | アクション開始/終了（Shift+キーで2チーム目） |

コントローラー右上の「?」アイコンからショートカットガイドを開けます。

## トラブルシューティング

- 映像が再生されない: MP4/MOV & H.264/AAC を推奨。権限/パスを確認。
- 音声同期がずれる: 両映像に音声があるか確認し、`同期リセット` → `音声同期を再実行`。手動同期/手動モードで微調整可能。
- タイムラインが反映されない: `timeline.json` の書き込み権限と残容量を確認。
- パッケージ読み込みエラー: `.metadata/config.json` の有無と相対パス変換を確認。
